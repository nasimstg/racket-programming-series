# Creating Graphics in Racket

**Blog Post:** [Read the full tutorial](https://learnhowto.vercel.app/blog/programming/schema/racket/creating-graphics-with-racket-example)

## Files

- shapes.rkt — Basic shapes, overlay, beside, above
- flower.rkt — Procedural flower from rotated ellipses
- animation.rkt — Animated growing circle

## Run

```bash
racket examples/graphics/animation.rkt
```
