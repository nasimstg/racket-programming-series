# Deployment Commands

```bash
# Compile a Racket file into a standalone executable
raco exe my-app.rkt

# Bundle the executable with all dependencies for distribution
raco distribute my-dist my-app.exe
```

See the [full blog post](https://learnhowto.vercel.app/blog/programming/schema/racket/deployment-and-distribution-shipping-racket-apps) for details.
